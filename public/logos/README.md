# Logos des compétences techniques

Ce dossier contient les logos SVG/PNG des technologies utilisées dans le composant SkillsGrid.

## Structure attendue

Les fichiers doivent être nommés selon le slug de chaque compétence :

- `html.svg` - Logo HTML
- `css.svg` - Logo CSS
- `javascript.svg` - Logo JavaScript
- `typescript.svg` - Logo TypeScript
- `react.svg` - Logo React
- `nextjs.svg` - Logo Next.js
- `tailwindcss.svg` - Logo Tailwind CSS
- `php.svg` - Logo PHP
- `symfony.svg` - Logo Symfony
- `git.svg` - Logo Git
- `mysql.svg` - Logo MySQL

## Format

- Format recommandé : SVG (vectoriel, scalable)
- Format alternatif : PNG (avec fond transparent si possible)
- Taille recommandée : 64x64px minimum pour une bonne qualité d'affichage

## Fallback

Si un logo n'est pas trouvé, le composant affichera le nom de la compétence en texte comme fallback.
